1. Aplikasi Web (Rekomendasi Utama)

Gunakan: Streamlit atau Flask (untuk web interaktif Python)

Kenapa?

  • Bisa langsung membaca input teks dari user.

  • Cocok untuk tugas akhir atau demo penelitian.

Contoh Sederhana dengan Streamlit

bash
pip install streamlit

Script Streamlit (app.py)

python
import streamlit as st
import pickle
import re
from Sastrawi.Stemmer.StemmerFactory import StemmerFactory

# Load model Naive Bayes yang sudah dilatih
model = pickle.load(open('model_nb.pkl', 'rb'))
vectorizer = pickle.load(open('vectorizer.pkl', 'rb'))

# Inisialisasi stemmer
factory = StemmerFactory()
stemmer = factory.create_stemmer()

def preprocessing(text):
    text = text.lower()
    text = re.sub(r'[^a-z\s]', '', text)
    text = stemmer.stem(text)
    return text

# UI
st.title("Analisis Sentimen Film 'Jumbo'")
user_input = st.text_area("Masukkan komentar Anda tentang film Jumbo:")

if st.button("Analisis"):
    if user_input.strip() != "":
        clean_text = preprocessing(user_input)
        text_vector = vectorizer.transform([clean_text])
        prediction = model.predict(text_vector)[0]

        if prediction == 1:
            st.success("Sentimen: **Positif** ")
        else:
            st.error("Sentimen: **Negatif** ")

1. Install Streamlit

2. Buat File Python (app.py)

Contoh kode ini menampilkan form untuk user memasukkan komentar, lalu memprosesnya dengan model Naive Bayes:

python
import streamlit as st
import pickle
import re
from Sastrawi.Stemmer.StemmerFactory import StemmerFactory

# Load model dan vectorizer
model = pickle.load(open('model_nb.pkl', 'rb'))          # ubah nama sesuai file model kamu
vectorizer = pickle.load(open('vectorizer.pkl', 'rb'))    # ubah nama sesuai file vectorizer kamu

# Inisialisasi stemmer
factory = StemmerFactory()
stemmer = factory.create_stemmer()

# Fungsi preprocessing teks
def preprocessing(text):
    text = text.lower()
    text = re.sub(r'[^a-z\s]', '', text)  # hilangkan karakter selain huruf dan spasi
    text = stemmer.stem(text)             # stemming
    return text

# UI Streamlit
st.title("Analisis Sentimen Film 'Jumbo'")
st.write("Masukkan komentar penonton film, lalu sistem akan memprediksi apakah sentimennya **positif** atau **negatif**.")

# Input teks dari user
user_input = st.text_area("Komentar pengguna:")

if st.button("Analisis Sentimen"):
    if user_input.strip() != "":
        clean_text = preprocessing(user_input)
        text_vector = vectorizer.transform([clean_text])
        prediction = model.predict(text_vector)[0]

        if prediction == 1:
            st.success(" Sentimen: **Positif**")
        else:
            st.error(" Sentimen: **Negatif**")

3. Jalankan Aplikasi Streamlit

Setelah kamu simpan file tersebut sebagai app.py, jalankan dengan perintah:

# app.py
import streamlit as st
import pickle
import re

# Load model dan vectorizer
model = pickle.load(open('model_nb.pkl', 'rb'))
vectorizer = pickle.load(open('vectorizer.pkl', 'rb'))

# Fungsi preprocessing sederhana

    text = text.lower()
    text = re.sub(r"[^a-z\s]", "", text)
    return text

# Judul Aplikasi

if st.button("Analisis"):
    cleaned = clean_text(user_input)
    vectorized = vectorizer.transform([cleaned])
    prediction = model.predict(vectorized)[0]

    if prediction == 1:
        st.success(" Sentimen: POSITIF")
    else:
        st.error(" Sentimen: NEGATIF")
